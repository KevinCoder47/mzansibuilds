# MzansiBuilds 🇿🇦 🚀
**Derivco Code Skills Challenge**

MzansiBuilds is a "Build in Public" platform for South African developers to showcase their projects, track progress, and collaborate.

## 📋 Features & User Journey
1.  **Account Management**: Developers can create and manage accounts.
2.  **Project Entries**: Create projects with specific stages and support requirements.
3.  **Live Feed**: Real-time view of what others are building with commenting and collaboration (hand-raise) features.
4.  **Milestones**: Update project progress continuously with achieved milestones.
5.  **Celebration Wall**: A hall of fame for completed projects.

## 🛠 Tech Stack
- **Frontend**: React (JavaScript) + Vite + Tailwind CSS
- **Backend**: Node.js + Express + TypeScript
- **Database**: PostgreSQL (Prisma ORM)
- **Real-time**: Socket.io (for the Live Feed)
- **Validation**: Zod

## 🏗 Project Structure
- `/client`: React frontend
- `/server`: Express backend (TypeScript)
- `/docs`: Documentation and UML diagrams
